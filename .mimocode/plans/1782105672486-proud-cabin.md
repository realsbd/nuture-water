# Fix: Product image not showing in hero section

## Root Cause
The `FlyingBottle` component (`components/FlyingBottle.tsx`) is fully implemented with scroll-based animation from `#bottle-start` to `#bottle-end`, but it is **never imported or rendered** in `app/page.tsx`. The anchor elements exist in `Herosection.tsx` and `AlkalineSection.tsx`, but nothing reads them.

## Fix
**File:** `app/page.tsx`

1. Add import for `FlyingBottle` component
2. Render `<FlyingBottle />` inside `<main>` (alongside `HeroSection` and `AlkalineSection`)

That's it. No other files need changes. The component already handles:
- Reading both anchor positions (`#bottle-start`, `#bottle-end`)
- Lerping position/size based on scroll progress
- Mid-flight flourish (rotation, scale, arc)
- Reduced motion support

## Verification
- Run `npm run dev` (or equivalent)
- Scroll the page — the product bottle should appear fixed and fly from the hero slot to the Alkaline section center
